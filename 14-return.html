退換貨服務申請

✦ 個人化設計訂製商品

根據消費者保護法第19條「客製化商品」不適用七天鑑賞期間（猶豫期）規定，恕無法以不喜歡設計或其他原因為由退貨、退款或換貨，請您慎重選購，避免影響日後選購權益。

✦ 一般商品

七日鑑賞期是根據消費者保護法規定，客戶（含本人、親友或管理員等）在簽收商品後隔日起算七日，商品保持全新狀態且完整包裝的情況下，可無條件退回商品，但逾期恕不受理。

  1. 根據消費者保護法規商品到貨享七天猶豫期之權益（注意！猶豫期非試用期）
  2. 辦理退貨商品必須是全新狀態（未經使用）且完整全新包裝，請勿缺漏任何配件，否則將會影響退貨權益。
  3. 本產品屬塑膠消耗品，若產品經使用過後表面產生刮痕或是掉漆等現象，恕不在退換貨規範內。

✦ 退換貨商品

聯名款、獨家款、周邊商品（個人化商品無法退換）買錯型號、更換圖款...等因素，將以退貨方式辦理 【申請退貨】 ，正確所需的商品請您直接重新下單訂購，節省商品換貨來回寄送所秏費的時間並較快能收到所需商品。

若有非一般商品退換貨的需求，如：寄錯商品 / 收到商品後因非人為因素之商品毀損、刮傷的瑕疵情況，請於七日鑑賞期內（完成取貨後隔日起計算）儘速填寫 【 客服表單 】 選擇＂產品問題＂通知本公司客服人員，我們將會與您確認訂單與商品瑕疵鑑定，並儘速協助您。

申請退貨（請點我）
  1. 請您於鑑賞期七日內申請【退貨服務】，持退貨便至超商辦理寄回，運費犀牛盾RHINOSHIELD 承擔（超過鑑賞期則需自行負擔費用寄回），正確的商品請您直接重新下單訂購，以節省貨物因來回寄送所秏費的時間。
  2. 退貨商品需保持完整，若商品經過人為不當拆卸使用而造成損毀、擦傷、刮傷髒污或包裝破損不完整，以致無法再銷售，或是配件不齊全者，請恕無法受理並安排寄回。（請勿直接將退貨貼紙黏貼於商品外包裝盒，並將商品妥善包裝，不負責寄回途中內容物掉出、遺失之商品金額。）
  3. 若退貨訂單有申請統編發票者，申請退貨後即已同意由 RHINOSHIELD 犀牛盾代為處理銷貨退回折讓單證明，您將收到電子折讓單通知信，屆時再煩請自行下載檔案印出【第三、四聯折讓單、蓋上發票章】並交付您的財會單位。
  4. 台灣訂單若退貨導致原訂單尚有商品保留金額未達免運費$400元標準，退款金額將直接抵扣運費$80元整。
  5. 若部分商品辦理退貨，僅保留加購商品時，加購商品將恢復原價計算。
  6. 若訂單符合【滿額優惠】活動，如退貨後保留商品金額未達滿額$2000門檻，滿額優惠折扣將從退款金額中扣除。
  7. 購買商品若退貨， 訂單活動折扣碼恕不再補發。
  8. 配送海外之包裹若拒收、退貨，退刷僅退商品金額，運費恕不退刷。

【注意事項】

  1. 每筆訂單僅提供乙次免費退貨服務，若超商寄回後需追加退貨商品，請自行負擔寄送運費。
  2. 請將商品妥善包裝，不負責寄回途中內容物掉出、遺失之商品金額。
  3. 若非台灣國內辦理退換貨需自行負擔退換貨之運費，其他海外國家建議直接由國外官方網站訂購，運費部分因每個國家收費不同，因此運費會於結帳頁面顯示。
  4. 【退款說明】為加速退款流程，款項一律直接退回原刷卡之信用卡/匯款帳戶
    • 信用卡付款：使用信用卡付款之訂單，退款將會刷退至當初刷卡付款的信用卡帳戶中，退款明細將隨下期帳單附上（退刷作業需14個工作天）
    • 黑貓宅配 / 超商貨到付款：請提供以下相關資訊【 銀行(台幣帳戶)、銀行分行、戶名、帳號，以上資訊缺一不可 】，資訊若提供錯誤造成二次匯款須扣除手續費(NT$30)，還請留意提供帳號正確與否。 (退款作業需14個工作天)。
    • 提供退款帳號請注意：提款卡卡號不一定等於存摺帳號，請以存摺資訊為準。
    • 一般銀行帳戶，請勿自行加入銀行代碼 ，帳號欄位請填寫存摺帳號即可，以免造成轉帳錯誤或失敗。
